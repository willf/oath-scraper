Oath ID 534: Homer, Odyssey, 5.177-187, (literary, Epic., date uncertain)


  Date: mythical time
  Reference type: Oath
  State: Ogygie   Location: unknown
 
  Swearer:
Calypso (female, mature, power-holder, n/a)
  Swearee:
Odysseus (male, mature, power-holder, Ithacan)
 
  Proposed by:
Odysseus
 
  If taken: unknown
  If refused: unknown
  If kept: unknown
  If broken: unknown
 
  Taken: yes   True: yes
  Impact: Odysseus feels able to leave in safety
  Consequences of breach: n/a
 
  Sanctifying features unknown
 
  Statement: [That she is not planning another painful trial for Odysseus]
οὐδ’ ἂν ἐγώ γ’ ἀέκητι σέθεν σχεδίης ἐπιβαίην, εἰ μή μοι τλαίης γε, θεά, μέγαν ὅρκον ὀμόσσαι
μή τί μοι αὐτῷ πῆμα κακὸν βουλευσέμεν ἄλλο.”
ὣς φάτο, μείδησεν δὲ Καλυψώ, δῖα θεάων,
χειρί τέ μιν κατέρεξεν ἔπος τ’ ἔφατ’ ἔκ τ’ ὀνόμαζεν·
“ἦ δὴ ἀλιτρός γ’ ἐσσὶ καὶ οὐκ ἀποφώλια εἰδώς,
οἷον δὴ τὸν μῦθον ἐπεφράσθης ἀγορεῦσαι.
ἴστω νῦν τόδε γαῖα καὶ οὐρανὸς εὐρὺς ὕπερθε
καὶ τὸ κατειβόμενον Στυγὸς ὕδωρ, ὅς τε μέγιστος
ὅρκος δεινότατός τε πέλει μακάρεσσι θεοῖσι,
μή τί τοι αὐτῷ πῆμα κακὸν βουλευσέμεν ἄλλο.
 
  Linguistic: istw, omos, ork
 
  God(s) invoked:
Gaia    
Ouranos Epithet(s) wide
Dripping water of the Styx    
 
  Remarks: It is noteworthy that Calyspso has already promised to send Odysseus on his way with good will at 5.160-170, but Odysseus feels the need to request an oath nonetheless.
