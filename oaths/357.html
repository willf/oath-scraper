Oath ID 357: IG II(2), 96, 19-20, (inscriptional, Public, 375/4)


  Date: 375/4 (375 to 374)
  Reference type: Oath
  State: Cephallenia   Location: unknown
 
  Swearer:
Representatives (male, mature/old, power-holder, Cephallenian)
  Swearee:
Three/five Athenians (male, mature/old, power-holder, Athenian)
 
  Proposed by:
Critias
 
  If taken: Their alliance would be formalised
  If refused: There would not be an alliance
  If kept: unknown
  If broken: unknown
 
  Taken: yes   True: unknown
  Impact: unknown
  Consequences of breach: unknown
 
  Sanctifying features unknown
 
  Statement: [? That they would abide by the following terms:
1. All allies were to be free and autonomous, with no garrisons or tribute required;
2. The Athenians would give up any cleruchies, and would not be permitted to occupy and land belonging to the allies;
3. The parties involved would defend each other if attacked]
 
  Linguistic: ork
 
  God(s) invoked:
unknown    
 
  Remarks: This inscription records an alliance between Athens and Cephallenia under the auspices of the so-called Second Athenian Confederacy.
The decree indicates that three or five Athenians were to be chosen to receive the oaths of the new allies (see also oath id 355 and 356). For the oath of the Athenians see oath id 353. For the oath of the existing allies of Athens see oath id 354.
For other references to the Second Athenian Confederacy see oath id 2974.
