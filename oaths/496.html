Oath ID 496: Aristophanes, Frogs, 1165-1166, (literary, Comic., 405)


  Date: 405
  Reference type: Oath
  State: Underworld   Location: House of Hades
 
  Swearer:
Dionysus (male, mature, power-holder, n/a)
  Swearee:
Aeschylus (male, old, free, Athenian)
Euripides (male, old, free, Athenian)
 
  Proposed by:
n/a
 
  If taken: n/a
  If refused: n/a
  If kept: unknown
  If broken: unknown
 
  Taken: yes   True: more complex
  Impact: Indicates that Aeschylus is now winning this argument
  Consequences of breach: unknown
 
  Sanctifying features none
 
  Statement: [That a man who is an exile both returns and remigrates]
φεύγων δ’ ἀνὴρ ἥκει τε καὶ κατέρχεται.
ΔΙ. Εὖ νὴ τὸν Ἀπόλλω
 
  Linguistic: nh ton
 
  God(s) invoked:
Apollo    
 
  Remarks: Aeschylus here rebuts Euripides' statement which had originally convinced Dionysus, who responded with an informal oath at id 494. The fact that Dionysus swears by Apollo here, as opposed to the more frequently invoked deity Zeus (called on in oath id 494), emphasizes Dionysus' turnabout in opinion. Swearing by Apollo is more emphatic in comedy than swearing by Zeus. But see also oath id 505, where Euripides trumps Aeschylus and Dionysus swears by Hermes.
